## 사전 준비

Cline CLI는 Node.js 20 이상이 필요합니다. 최상의 경험을 위해 Node.js 22 사용을 권장합니다.

Node.js 버전 확인:

```bash
node --version
```

## 설치

```bash
npm install -g cline
```

설치 후 Cline 계정으로 인증하세요:

```bash
cline auth
```

이 명령은 로그인 및 선호하는 AI 모델 제공자 구성을 위한 인증 마법사를 시작합니다.

## 빠른 시작

몇 초 만에 Cline을 시작하세요:

```bash
cline
```

끝입니다! 어떤 디렉터리에서든 `cline`을 실행하면 대화형 세션이 시작되어 AI 에이전트와 대화할 수 있습니다. 작업을 입력하고 계획을 확인한 뒤, 실행할 준비가 되면 `/act`를 입력하세요.

상호작용 없이 더 빠르게 실행하려면:

```bash
cline "Add unit tests to utils.js"
```

단일 명령으로 Cline을 실행하므로 빠른 작업이나 자동화에 적합합니다.
