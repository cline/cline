# 技术栈

## 使用的技术
- 核心语言：TypeScript
- 前端框架：React
- 运行时：Node.js
- 扩展框架：VSCode Extension API
- 状态管理：React Context
- 构建工具：esbuild, webpack
- 测试框架：Mocha, Chai
- 代码质量：ESLint, Prettier
- 依赖管理：npm
- 文档工具：Markdown

## 开发环境设置
- Node.js v18+
- npm 8+
- VSCode 1.85+
- 推荐扩展：
  - ESLint
  - Prettier
  - TypeScript工具链
- 环境变量：
  - API_KEY：用于第三方API认证
  - DEBUG：调试模式开关

## 技术限制
1. 仅支持VSCode环境
2. 依赖Node.js运行时
3. 浏览器自动化功能受CORS限制
4. 内存使用受VSCode扩展限制
5. 并发任务数量有限制
6. 不支持离线模式
