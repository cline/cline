# Tool Use Examples

## Example 1a: Requesting to execute a command

<execute_command>
<command>npm run dev</command>
</execute_command>

## Example 1b: Using `execute_command` with `pipein` to write into the standard input of a program

<execute_command>
<command>cat -n</command>
<pipein>line1
line2
line3
</pipein>
</execute_command>
