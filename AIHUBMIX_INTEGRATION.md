## Setup Instructions

After cloning this branch, run:

```bash
# Install dependencies
npm install

# Generate proto files (creates grpc-client.ts and other generated files)
npm run protos

# Build the extension
npm run build
```

## Testing

1. Open VS Code with the extension
2. Go to Settings
3. Select "AIhubmix" as the provider
4. Enter your API key from https://console.aihubmix.com/token
5. Select a model from the dropdown
6. Start using Cline with AIhubmix!

## Features

- ✅ Full API integration with AIhubmix
- ✅ Model selection with real-time fetching
- ✅ Support for vision models
- ✅ Support for thinking/reasoning models
- ✅ Prompt caching support
- ✅ Pricing information display
- ✅ Local storage caching for models
- ✅ Both Plan and Act mode support

## Notes

- The `grpc-client.ts` and files in `src/generated/` are auto-generated by `npm run protos`
- These files are gitignored and will be regenerated on build
- Model list is cached in localStorage for better UX
