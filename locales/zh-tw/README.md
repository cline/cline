認識 Ponder —— 一個將 VS Code 轉變為創意寫作工作室的 AI 寫作助手。

Ponder 是基於強大的 Cline 專案的分支，將 AI 能力從軟體開發重新定向到創意寫作、內容創作和文學創作領域。借助 Claude 3.7 Sonnet 等先進語言模型，Ponder 能夠協助處理複雜的寫作任務，包括小說創作、詩歌寫作、劇本開發、學術寫作、內容行銷和編輯工作。

1. **開始您的創作專案**：描述您的寫作目標——無論是小說、劇本、部落格文章還是行銷文案。
2. **Ponder 分析您的上下文**：助手會檢查您現有的文件、研究資料和寫作風格。
3. **全面的寫作支援**：創建和編輯文件，研究主題，組織內容結構，提供編輯回饋。
4. **審查和完善**：當寫作任務完成時，Ponder 會呈現結果供您審查和修訂。

> [!TIP]
> 使用 `CMD/CTRL + Shift + P` 快捷鍵開啟命令面板並輸入 "Ponder: Open In New Tab" 將擴充功能作為標籤在編輯器中開啟。

## 主要功能

### 使用任何 API 和模型
Ponder 支援 OpenRouter、Anthropic、OpenAI、Google Gemini、AWS Bedrock 等多種 API 提供商。

### 研究和網頁瀏覽
借助 Claude 3.5 Sonnet 的電腦使用功能，Ponder 可以啟動瀏覽器進行研究、事實查核和內容收集。

### 創建和編輯文件
直接在編輯器中創建和編輯文件，提供差異檢視，監控寫作品質和風格一致性。

### 自訂寫作工具
透過模型上下文協定擴展功能，創建專門的寫作工具如角色資料庫、風格分析器等。

## 開始使用

### 安裝
1. 從 [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev) 安裝 Ponder
2. 配置您偏好的 AI 模型
3. 開始您的第一個寫作專案

### 寫作用例
- **創意寫作**：小說、短篇故事、詩歌、劇本
- **內容行銷**：部落格文章、社群媒體內容、行銷文案
- **學術寫作**：研究論文、論文、學位論文工作
- **技術寫作**：文件、手冊、指南
- **新聞報導**：文章、採訪、調查報導
- **商務寫作**：報告、提案、簡報
