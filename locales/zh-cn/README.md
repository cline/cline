认识 Ponder —— 一个将 VS Code 转变为创意写作工作室的 AI 写作助手。

Ponder 是基于强大的 Cline 项目的分支，将 AI 能力从软件开发重新定向到创意写作、内容创作和文学创作领域。借助 Claude 3.7 Sonnet 等先进语言模型，Ponder 能够协助处理复杂的写作任务，包括小说创作、诗歌写作、剧本开发、学术写作、内容营销和编辑工作。

1. **开始您的创作项目**：描述您的写作目标——无论是小说、剧本、博客文章还是营销文案。
2. **Ponder 分析您的上下文**：助手会检查您现有的文档、研究资料和写作风格。
3. **全面的写作支持**：创建和编辑文档，研究主题，组织内容结构，提供编辑反馈。
4. **审查和完善**：当写作任务完成时，Ponder 会呈现结果供您审查和修订。

> [!TIP]
> 使用 `CMD/CTRL + Shift + P` 快捷键打开命令面板并输入 "Ponder: Open In New Tab" 将扩展作为标签在编辑器中打开。

## 主要功能

### 使用任何 API 和模型
Ponder 支持 OpenRouter、Anthropic、OpenAI、Google Gemini、AWS Bedrock 等多种 API 提供商。

### 研究和网页浏览
借助 Claude 3.5 Sonnet 的计算机使用功能，Ponder 可以启动浏览器进行研究、事实核查和内容收集。

### 创建和编辑文档
直接在编辑器中创建和编辑文档，提供差异视图，监控写作质量和风格一致性。

### 自定义写作工具
通过模型上下文协议扩展功能，创建专门的写作工具如角色数据库、风格分析器等。

## 开始使用

### 安装
1. 从 [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev) 安装 Ponder
2. 配置您首选的 AI 模型
3. 开始您的第一个写作项目

### 写作用例
- **创意写作**：小说、短篇故事、诗歌、剧本
- **内容营销**：博客文章、社交媒体内容、营销文案
- **学术写作**：研究论文、论文、学位论文工作
- **技术写作**：文档、手册、指南
- **新闻报道**：文章、采访、调查报道
- **商务写作**：报告、提案、演示文稿
