Conheça o Ponder —— um assistente de escrita com IA que transforma o VS Code em seu estúdio de escrita criativa.

Ponder é um fork do poderoso projeto Cline, redirecionando as capacidades de IA do desenvolvimento de software para a escrita criativa, criação de conteúdo e composição literária. Com modelos de linguagem avançados como Claude 3.7 Sonnet, Ponder pode auxiliar em tarefas complexas de escrita incluindo escrita de romances, criação de poesia, desenvolvimento de roteiros, escrita acadêmica, marketing de conteúdo e trabalho editorial.

1. **Inicie seu projeto criativo**: Descreva seu objetivo de escrita — seja um romance, roteiro, artigo de blog ou copy de marketing.
2. **Ponder analisa seu contexto**: O assistente revisa seus documentos existentes, materiais de pesquisa e estilo de escrita.
3. **Suporte abrangente de escrita**: Criar e editar documentos, pesquisar tópicos, organizar estruturas de conteúdo, fornecer feedback editorial.
4. **Revisar e refinar**: Quando uma tarefa de escrita é concluída, Ponder apresenta os resultados para revisão e refinamento.

> [!TIP]
> Use o atalho `CMD/CTRL + Shift + P` para abrir a paleta de comandos e digite "Ponder: Open In New Tab" para abrir a extensão como uma aba no seu editor.

## Recursos Principais

### Usar qualquer API e Modelo
Ponder suporta vários provedores de API como OpenRouter, Anthropic, OpenAI, Google Gemini, AWS Bedrock e mais.

### Pesquisa e Navegação Web
Com a funcionalidade de uso de computador do Claude 3.5 Sonnet, Ponder pode lançar navegadores para realizar pesquisas, verificação de fatos e coleta de conteúdo.

### Criar e Editar Documentos
Criar e editar documentos diretamente no editor, com visualização de diferenças e monitoramento de qualidade de escrita e consistência de estilo.

### Ferramentas de Escrita Personalizadas
Estender funcionalidades através do Protocolo de Contexto de Modelo e criar ferramentas especializadas de escrita como bancos de dados de personagens, analisadores de estilo, etc.

## Começando

### Instalação
1. Instale o Ponder do [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev)
2. Configure seu modelo de IA preferido
3. Inicie seu primeiro projeto de escrita

### Casos de Uso de Escrita
- **Escrita Criativa**: Romances, contos, poesia, roteiros
- **Marketing de Conteúdo**: Artigos de blog, conteúdo de mídia social, copy de marketing
- **Escrita Acadêmica**: Trabalhos de pesquisa, ensaios, trabalho de tese
- **Escrita Técnica**: Documentação, manuais, guias
- **Jornalismo**: Artigos, entrevistas, reportagens investigativas
- **Escrita Empresarial**: Relatórios, propostas, apresentações
