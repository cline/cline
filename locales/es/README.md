Conoce Ponder —— un asistente de escritura con IA que transforma VS Code en tu estudio de escritura creativa.

Ponder es un fork del poderoso proyecto Cline, redirigiendo las capacidades de IA desde el desarrollo de software hacia la escritura creativa, creación de contenido y composición literaria. Con modelos de lenguaje avanzados como Claude 3.7 Sonnet, Ponder puede asistir con tareas complejas de escritura incluyendo escritura de novelas, creación de poesía, desarrollo de guiones, escritura académica, marketing de contenidos y trabajo editorial.

1. **Inicia tu proyecto creativo**: Describe tu objetivo de escritura — ya sea una novela, guión, artículo de blog o copy de marketing.
2. **Ponder analiza tu contexto**: El asistente revisa tus documentos existentes, materiales de investigación y estilo de escritura.
3. **Soporte integral de escritura**: Crear y editar documentos, investigar temas, organizar estructuras de contenido, proporcionar retroalimentación editorial.
4. **Revisar y refinar**: Cuando se completa una tarea de escritura, Ponder presenta los resultados para revisión y revisión.

> [!TIP]
> Usa el atajo `CMD/CTRL + Shift + P` para abrir la paleta de comandos y escribe "Ponder: Open In New Tab" para abrir la extensión como una pestaña en tu editor.

## Características Principales

### Usar cualquier API y Modelo
Ponder soporta varios proveedores de API como OpenRouter, Anthropic, OpenAI, Google Gemini, AWS Bedrock y más.

### Investigación y Navegación Web
Con la función de uso de computadora de Claude 3.5 Sonnet, Ponder puede lanzar navegadores para realizar investigación, verificación de hechos y recopilación de contenido.

### Crear y Editar Documentos
Crear y editar documentos directamente en el editor, con vista de diferencias y monitoreo de calidad de escritura y consistencia de estilo.

### Herramientas de Escritura Personalizadas
Extender funcionalidades a través del Protocolo de Contexto de Modelo y crear herramientas especializadas de escritura como bases de datos de personajes, analizadores de estilo, etc.

## Comenzar

### Instalación
1. Instala Ponder desde el [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev)
2. Configura tu modelo de IA preferido
3. Inicia tu primer proyecto de escritura

### Casos de Uso de Escritura
- **Escritura Creativa**: Novelas, cuentos cortos, poesía, guiones
- **Marketing de Contenidos**: Artículos de blog, contenido de redes sociales, copy de marketing
- **Escritura Académica**: Trabajos de investigación, ensayos, trabajo de tesis
- **Escritura Técnica**: Documentación, manuales, guías
- **Periodismo**: Artículos, entrevistas, reportajes de investigación
- **Escritura Empresarial**: Informes, propuestas, presentaciones
