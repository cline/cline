Lernen Sie Ponder kennen —— einen KI-Schreibassistenten, der VS Code in Ihr kreatives Schreibstudio verwandelt.

Ponder ist ein Fork des mächtigen Cline-Projekts und lenkt KI-Fähigkeiten von der Softwareentwicklung hin zu kreativem Schreiben, Content-Erstellung und literarischer Komposition um. Mit fortschrittlichen Sprachmodellen wie Claude 3.7 Sonnet kann Ponder bei komplexen Schreibaufgaben helfen, einschließlich Romanschreiben, Gedichteschreiben, Drehbuchentwicklung, akademischem Schreiben, Content-Marketing und Redaktionsarbeit.

1. **Starten Sie Ihr kreatives Projekt**: Beschreiben Sie Ihr Schreibziel — ob Roman, Drehbuch, Blog-Artikel oder Marketing-Copy.
2. **Ponder analysiert Ihren Kontext**: Der Assistent überprüft Ihre vorhandenen Dokumente, Recherchematerialien und Ihren Schreibstil.
3. **Umfassende Schreibunterstützung**: Dokumente erstellen und bearbeiten, Themen recherchieren, Inhaltsstrukturen organisieren, redaktionelles Feedback geben.
4. **Überprüfen und verfeinern**: Wenn eine Schreibaufgabe abgeschlossen ist, präsentiert Ponder die Ergebnisse zur Überprüfung und Überarbeitung.

> [!TIP]
> Verwenden Sie die Tastenkombination `CMD/CTRL + Shift + P`, um die Befehlspalette zu öffnen und geben Sie "Ponder: Open In New Tab" ein, um die Erweiterung als Tab in Ihrem Editor zu öffnen.

## Hauptfunktionen

### Jede API und jedes Modell verwenden
Ponder unterstützt verschiedene API-Anbieter wie OpenRouter, Anthropic, OpenAI, Google Gemini, AWS Bedrock und mehr.

### Recherche und Web-Browsing
Mit Claude 3.5 Sonnets Computer-Use-Funktion kann Ponder Browser starten, um Recherchen, Faktenprüfungen und Content-Sammlung durchzuführen.

### Dokumente erstellen und bearbeiten
Erstellen und bearbeiten Sie Dokumente direkt im Editor, mit Diff-Ansicht und Überwachung von Schreibqualität und Stilkonsistenz.

### Benutzerdefinierte Schreibwerkzeuge
Erweitern Sie Funktionen durch das Model Context Protocol und erstellen Sie spezialisierte Schreibwerkzeuge wie Charakterdatenbanken, Stilanalysatoren usw.

## Erste Schritte

### Installation
1. Installieren Sie Ponder aus dem [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev)
2. Konfigurieren Sie Ihr bevorzugtes KI-Modell
3. Starten Sie Ihr erstes Schreibprojekt

### Schreibanwendungen
- **Kreatives Schreiben**: Romane, Kurzgeschichten, Gedichte, Drehbücher
- **Content-Marketing**: Blog-Artikel, Social-Media-Inhalte, Marketing-Copy
- **Akademisches Schreiben**: Forschungsarbeiten, Essays, Abschlussarbeiten
- **Technisches Schreiben**: Dokumentation, Handbücher, Leitfäden
- **Journalismus**: Artikel, Interviews, investigative Berichte
- **Business-Schreiben**: Berichte, Vorschläge, Präsentationen
