تعرف على Ponder —— مساعد الكتابة بالذكاء الاصطناعي الذي يحول VS Code إلى استوديو الكتابة الإبداعية الخاص بك.

Ponder هو تفرع من مشروع Cline القوي، يعيد توجيه قدرات الذكاء الاصطناعي من تطوير البرمجيات إلى مجال الكتابة الإبداعية وإنشاء المحتوى والتأليف الأدبي. باستخدام نماذج اللغة المتقدمة مثل Claude 3.7 Sonnet، يمكن لـ Ponder المساعدة في مهام الكتابة المعقدة بما في ذلك كتابة الروايات وإنشاء الشعر وتطوير السيناريوهات والكتابة الأكاديمية وتسويق المحتوى والعمل التحريري.

1. **ابدأ مشروعك الإبداعي**: صف هدف الكتابة الخاص بك — سواء كان رواية أو سيناريو أو مقال مدونة أو نسخة تسويقية.
2. **Ponder يحلل السياق الخاص بك**: يراجع المساعد وثائقك الموجودة ومواد البحث وأسلوب الكتابة.
3. **دعم شامل للكتابة**: إنشاء وتحرير الوثائق، البحث في المواضيع، تنظيم هياكل المحتوى، تقديم التغذية الراجعة التحريرية.
4. **المراجعة والتحسين**: عند اكتمال مهمة الكتابة، يقدم Ponder النتائج للمراجعة والتنقيح.

> [!TIP]
> استخدم الاختصار `CMD/CTRL + Shift + P` لفتح لوحة الأوامر واكتب "Ponder: Open In New Tab" لفتح الإضافة كعلامة تبويب في محررك.

## الميزات الرئيسية

### استخدام أي API ونموذج
يدعم Ponder مختلف مقدمي API مثل OpenRouter وAnthropic وOpenAI وGoogle Gemini وAWS Bedrock والمزيد.

### البحث وتصفح الويب
مع وظيفة استخدام الكمبيوتر في Claude 3.5 Sonnet، يمكن لـ Ponder تشغيل المتصفحات لإجراء البحوث والتحقق من الحقائق وجمع المحتوى.

### إنشاء وتحرير الوثائق
إنشاء وتحرير الوثائق مباشرة في المحرر، مع عرض الاختلافات ومراقبة جودة الكتابة واتساق الأسلوب.

### أدوات الكتابة المخصصة
توسيع الوظائف من خلال بروتوكول سياق النموذج وإنشاء أدوات كتابة متخصصة مثل قواعد بيانات الشخصيات ومحللات الأسلوب وغيرها.

## البداية

### التثبيت
1. قم بتثبيت Ponder من [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=saoudrizwan.claude-dev)
2. قم بتكوين نموذج الذكاء الاصطناعي المفضل لديك
3. ابدأ مشروع الكتابة الأول

### حالات استخدام الكتابة
- **الكتابة الإبداعية**: الروايات، القصص القصيرة، الشعر، السيناريوهات
- **تسويق المحتوى**: مقالات المدونة، محتوى وسائل التواصل الاجتماعي، النسخ التسويقية
- **الكتابة الأكاديمية**: أوراق البحث، المقالات، أعمال الأطروحة
- **الكتابة التقنية**: الوثائق، الأدلة، الإرشادات
- **الصحافة**: المقالات، المقابلات، التقارير الاستقصائية
- **الكتابة التجارية**: التقارير، المقترحات، العروض التقديمية
