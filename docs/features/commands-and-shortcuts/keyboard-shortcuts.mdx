Cline's keyboard shortcuts let you access AI assistance without taking your hands off the keyboard. Speed up your workflow by using hotkeys for common Cline actions.

## Default Keyboard Shortcuts

Cline comes with these built-in keyboard shortcuts:

| Action           | Windows/Linux | macOS   | When Active                  | Description                        |
| ---------------- | ------------- | ------- | ---------------------------- | ---------------------------------- |
| Add to Cline     | `Ctrl+'`      | `Cmd+'` | When text is selected        | Adds selected code to Cline chat   |
| Focus Chat Input | `Ctrl+'`      | `Cmd+'` | When no text is selected     | Focuses the Cline chat input field |
| Generate Commit  | (unset)       | (unset) | When Git is the SCM provider | Generate commit message            |

The `Ctrl+'` / `Cmd+'` shortcut is context-aware: it adds selected code to chat when you have text selected, or focuses the chat input when nothing is selected.

## Commands NOT Available for Keyboard Shortcuts

Some Cline features are **only accessible through other UI interactions** and cannot be assigned keyboard shortcuts:

### Code Actions (Lightbulb Menu Only)

| Command                  | How to Access                                                   | Description                                              |
| ------------------------ | --------------------------------------------------------------- | -------------------------------------------------------- |
| **Fix with Cline**       | Click the lightbulb icon when there are errors in your code     | Automatically fixes errors and warnings with AI          |

**Why can't I assign a shortcut to "Fix with Cline"?**  
This command is implemented as a VSCode Code Action, which only appears contextually in the lightbulb menu when there are diagnostics (errors/warnings) in your code. It requires the error context to work properly, so it's not available as a standalone keyboard command.

**Alternative workflows:**
- Use `Ctrl+'` / `Cmd+'` to add code with errors to chat, then ask Cline to fix them
- Click the lightbulb icon (⚡) that appears next to lines with errors
- Right-click on code and select "Add to Cline" to discuss issues

## How to Customize Keyboard Shortcuts

Customize any of the available commands to match your preferences:

1. **Open the Keyboard Shortcuts editor:**
   - Press `Ctrl+K Ctrl+S` (Windows/Linux) or `Cmd+K Cmd+S` (macOS)
   - Or go to **File > Preferences > Keyboard Shortcuts**

2. **Search for "Cline"** to see all available commands

3. **Click the pencil icon** (✏️) next to the command you want to customize

4. **Press the key combination** you want to use

5. **Press Enter** to save

### Tips for Choosing Shortcuts

- **Avoid conflicts:** Make sure your shortcut doesn't override important VSCode shortcuts
- **Use modifiers:** Combine `Ctrl`/`Cmd`, `Alt`, and `Shift` to avoid conflicts
- **Be ergonomic:** Choose shortcuts that are comfortable for your keyboard layout
- **Create patterns:** Use similar shortcuts for related commands (e.g., all `Alt+` combinations)

## Keyboard-First Workflow Example

Here's how you can use Cline entirely with the keyboard:

1. **Select code:** Use VSCode's selection shortcuts (`Shift+Arrow`, `Ctrl+L` / `Cmd+L` for line)
2. **Send to Cline:** Press `Ctrl+'` / `Cmd+'`
3. **Type your question:** Focus is automatically on the chat input
4. **Submit:** Press `Enter` to send
5. **Apply changes:** Use `Tab` to navigate and `Enter` to approve

## Best Practices

- **Start with defaults:** Learn the default `Ctrl+'` / `Cmd+'` shortcut first - it's versatile and handles the most common use cases
- **Add gradually:** Don't create 20 shortcuts at once. Add them as you discover which commands you use frequently
- **Document your shortcuts:** Keep a note of custom shortcuts you create, especially if they're non-standard
- **Muscle memory:** Use shortcuts consistently for a few days to build muscle memory
- **Review periodically:** Revisit your shortcuts every month to optimize based on actual usage

Keyboard shortcuts may seem like a small optimization, but they can significantly speed up your workflow when using Cline regularly. By keeping your hands on the keyboard, you maintain your coding flow while still getting AI assistance exactly when you need it.
