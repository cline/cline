Web tools let Cline search the internet and pull in content from specific URLs while it works — useful for doc lookups, recent releases, or research that goes beyond your codebase and the model's training data.

Two tools are available:

- `web_search` — search the web and return relevant pages for a query
- `web_fetch` — fetch and analyze content from a specific URL

Cline calls these on its own when a task benefits from web context.

<Warning>
Web tools require the **Cline provider**. They are not available when using other providers like OpenRouter, Anthropic, AWS Bedrock, etc.
</Warning>

## Enabling Web Tools

1. Sign in to Cline and select the Cline provider
2. Toggle **Web Tools** under Feature Settings

<Note>
Web tools can be auto-approved with the "Use the browser" setting in [Auto Approve](/features/auto-approve).
</Note>

<Tip>
Building on the SDK? Web search and fetch are available as a plugin example. See [SDK Plugins](/sdk/plugins) and [Writing Plugins](/sdk/guides/writing-plugins).
</Tip>
