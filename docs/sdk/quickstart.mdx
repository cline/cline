This guide creates a minimal `Agent`, adds one custom tool, and shows when to switch to `ClineCore`.

## Prerequisites

- Node.js 22 or later
- An API key from a supported provider

## Setup

<Steps>
  <Step title="Create a new project">
    ```bash
    mkdir my-agent && cd my-agent
    npm init -y
    npm install @clinebot/agents @clinebot/shared tsx
    ```
  </Step>

  <Step title="Create your first agent">
    Create `agent.ts`:

    ```typescript
    import { Agent } from "@clinebot/agents"

    const agent = new Agent({
      providerId: "anthropic",
      modelId: "claude-sonnet-4-6",
      apiKey: process.env.ANTHROPIC_API_KEY,
      systemPrompt: "You are a helpful coding assistant. Be concise.",
      tools: [],
    })

    agent.subscribe((event) => {
      if (event.type === "assistant-text-delta") {
        process.stdout.write(event.text)
      }
    })

    const result = await agent.run("What is the fibonacci sequence? Show me a TypeScript implementation.")
    console.log("\n\nStatus:", result.status)
    console.log("Iterations:", result.iterations)
    console.log("Tokens used:", result.usage.inputTokens + result.usage.outputTokens)
    ```
  </Step>

  <Step title="Run it">
    ```bash
    ANTHROPIC_API_KEY=sk-ant-... npx tsx agent.ts
    ```
  </Step>
</Steps>

## Add a Custom Tool

```typescript
import { Agent } from "@clinebot/agents"
import type { AgentTool } from "@clinebot/shared"

const weatherTool: AgentTool<{ city: string }, { temperature: number; condition: string; city: string }> = {
  name: "get_weather",
  description: "Get the current weather for a city.",
  inputSchema: {
    type: "object",
    properties: {
      city: { type: "string", description: "City name" },
    },
    required: ["city"],
  },
  execute: async (input) => {
    return {
      output: { temperature: 72, condition: "sunny", city: input.city },
    }
  },
}

const agent = new Agent({
  providerId: "anthropic",
  modelId: "claude-sonnet-4-6",
  apiKey: process.env.ANTHROPIC_API_KEY,
  systemPrompt: "You are a weather assistant. Use the get_weather tool to answer questions.",
  tools: [weatherTool],
})

agent.subscribe((event) => {
  if (event.type === "assistant-text-delta") {
    process.stdout.write(event.text)
  }
  if (event.type === "tool-started") {
    console.log(`\n[calling tool: ${event.toolCall.toolName}]`)
  }
})

await agent.run("What's the weather in San Francisco?")
```

## Multi-Turn Conversations

Use `continue()` to keep a conversation going with the same agent:

```typescript
await agent.run("My name is Sarah and I'm building a REST API in Go.")
const result = await agent.continue("What framework would you recommend?")
console.log(result.outputText)
```

## Use ClineCore for Sessions and Built-In Tools

Install core:

```bash
npm install @clinebot/core
```

Then create a session:

```typescript
import { ClineCore } from "@clinebot/core"

const cline = await ClineCore.create({ clientName: "my-app" })

const session = await cline.start({
  prompt: "Create a README.md for this project",
  config: {
    providerId: "anthropic",
    modelId: "claude-sonnet-4-6",
    apiKey: process.env.ANTHROPIC_API_KEY,
    systemPrompt: "You are a helpful coding assistant.",
    cwd: process.cwd(),
    enableTools: true,
    enableSpawnAgent: false,
    enableAgentTeams: false,
  },
})

console.log("Session ID:", session.sessionId)
console.log("Result:", session.result?.text)
console.log("Finish reason:", session.result?.finishReason)

await cline.dispose()
```

ClineCore can enable built-in tools such as `read_files`, `search_codebase`, `run_commands`, `fetch_web_content`, `apply_patch`, `editor`, `skills`, `ask_question`, and `submit_and_exit`.

## Common Patterns

### Streaming CLI Output with Agent

Subscribe to `assistant-text-delta` events when you want to print model output as it streams:

```typescript
import { Agent } from "@clinebot/agents"

const agent = new Agent({
  providerId: "anthropic",
  modelId: "claude-sonnet-4-6",
  apiKey: process.env.ANTHROPIC_API_KEY,
  systemPrompt: "You are a concise coding assistant.",
  tools: [],
})

agent.subscribe((event) => {
  if (event.type === "assistant-text-delta") {
    process.stdout.write(event.text)
  }
})

await agent.run("Explain this repository structure")
```

### Persistent Session with ClineCore

Use `ClineCore` when you need a session ID that can receive follow-up prompts:

```typescript
import { ClineCore } from "@clinebot/core"

const cline = await ClineCore.create({ clientName: "example" })

const session = await cline.start({
  prompt: "Summarize this codebase",
  config: {
    providerId: "anthropic",
    modelId: "claude-sonnet-4-6",
    apiKey: process.env.ANTHROPIC_API_KEY,
    systemPrompt: "You are a helpful coding assistant.",
    cwd: process.cwd(),
    enableTools: true,
    enableSpawnAgent: false,
    enableAgentTeams: false,
  },
})

await cline.send({
  sessionId: session.sessionId,
  prompt: "Now identify risky files",
})

await cline.dispose()
```
