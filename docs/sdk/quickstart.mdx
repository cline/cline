This guide gets you from zero to a running agent. By the end, you'll have an agent that streams responses and calls custom tools.

## Prerequisites

- Node.js 22 or later
- An API key from a supported provider

## Setup

<Steps>
  <Step title="Create a new project">
    ```bash
    mkdir my-agent && cd my-agent
    npm init -y
    npm install @cline/sdk
    ```
  </Step>

  <Step title="Create your first agent">
    Create `agent.ts`:

    ```typescript
    import { Agent } from "@cline/sdk"

    const agent = new Agent({
      providerId: "anthropic",
      modelId: "claude-sonnet-4-6",
      apiKey: process.env.ANTHROPIC_API_KEY,
      maxIterations: 1,
    })

    agent.subscribe((event) => {
      if (event.type === "assistant-text-delta") {
        process.stdout.write(event.text)
      }
    })

    const result = await agent.run("What is the fibonacci sequence? Show me a TypeScript implementation.")
    console.log("\n\nStatus:", result.status)
    console.log("Tokens used:", result.usage.inputTokens + result.usage.outputTokens)
    ```

    <Note>
      This is the same pattern used in the [quickstart example](https://github.com/cline/sdk/tree/main/apps/examples/quickstart).
    </Note>
  </Step>

  <Step title="Run it">
    ```bash
    ANTHROPIC_API_KEY=sk-ant-... npx tsx agent.ts
    ```
  </Step>
</Steps>

## Add a Custom Tool

Tools let the agent take actions. Use `createTool` with a zod schema to define type-safe tools:

```typescript
import { Agent, createTool } from "@cline/sdk"
import { z } from "zod"

const agent = new Agent({
  providerId: "anthropic",
  modelId: "claude-sonnet-4-6",
  apiKey: process.env.ANTHROPIC_API_KEY,
  systemPrompt: "You are a helpful assistant. Use tools when needed.",
  tools: [
    createTool({
      name: "get_weather",
      description: "Get the current weather for a city.",
      inputSchema: z.object({
        city: z.string().describe("City name"),
      }),
      async execute(input) {
        return { temperature: 72, condition: "sunny", city: input.city }
      },
    }),
  ],
})
```

For a complete interactive agent with tools and streaming, see the [cli-agent example](https://github.com/cline/sdk/tree/main/apps/examples/cli-agent).

## Multi-Turn Conversations

Use `continue()` to keep a conversation going with the same agent:

```typescript
await agent.run("My name is Sarah and I'm building a REST API in Go.")
const result = await agent.continue("What framework would you recommend?")
console.log(result.outputText)
```

## Use ClineCore for Sessions and Built-In Tools

`ClineCore` adds session persistence, built-in tools (file read/write, shell, search, web fetch), and hub support:

```typescript
import { ClineCore } from "@cline/sdk"

const cline = await ClineCore.create({ clientName: "my-app" })

const session = await cline.start({
  prompt: "Create a README.md for this project",
  config: {
    providerId: "anthropic",
    modelId: "claude-sonnet-4-6",
    apiKey: process.env.ANTHROPIC_API_KEY,
    systemPrompt: "You are a helpful coding assistant.",
    cwd: process.cwd(),
    enableTools: true,
    enableSpawnAgent: false,
    enableAgentTeams: false,
  },
})

console.log("Session ID:", session.sessionId)
await cline.dispose()
```

See [Agent vs ClineCore](/sdk/runtime) for when to use each.
