Ready to see Beadsmith in action? This hands-on tutorial will walk you through building a website in under a minute. You'll experience how Beadsmith understands your requirements, creates files, and iterates on your feedbackâ€”all through natural conversation.

By the end of this guide, you'll have built a working website and learned the fundamentals of working with Beadsmith.

## Prerequisites

- **Beadsmith installed** in your editor ([Install Guide](/getting-started/installing-beadsmith))
- **AI model selected** ([Model Setup](/getting-started/selecting-your-model))
- **Any folder open** in your editor (or create a new empty folder)

## Step 1: Open Beadsmith

Click the Beadsmith icon in your editor's sidebar (left side). The chat panel will open.

<Info>
**Quick Tip:** You can also use `Cmd+Shift+P` (Mac) or `Ctrl+Shift+P` (Windows/Linux) and search for "Beadsmith: Open In New Tab"
</Info>

## Step 2: Give Beadsmith a Task

Copy and paste this prompt into Beadsmith's chat:

```
Create a simple website in a single HTML file. It should have:
- A welcome message saying "Hello from Beadsmith!"
- A colorful gradient background
- A button that cycles through different color themes when clicked
- Modern, clean design
- All CSS and JavaScript should be included in the same HTML file
```

<Frame>
  <img src="https://storage.googleapis.com/beadsmith_public_images/chat-prompt.png" alt="Beadsmith Chat Prompt"/>
</Frame>

Press Enter and watch Beadsmith work!

## Step 3: What Happens Next

Beadsmith will:

1. **Create a single file:**
   - `index.html` - A complete webpage with embedded CSS and JavaScript

2. **Ask for approval** (unless you've enabled auto-approve)
   - Click "Approve" to let Beadsmith create the file
   - You can review what it plans to do first

3. **Complete the task** within seconds

## Step 4: View Your Website

Once Beadsmith finishes:

1. **Find `index.html`** in your editor's file explorer
2. **Right-click it** and select:
   - "Reveal in Finder/Explorer" then double-click to open in your browser
3. **Click the button** to see the color themes change!

## Try Making Changes

In the same chat, try asking:

```
Add a counter that shows how many times the button has been clicked
```

or

```
Make the welcome message fade in when the page loads
```

Beadsmith understands the context from your previous conversation and will update the file accordingly.

**You now know how to:**
- Give Beadsmith a task with a clear prompt
- Review and approve Beadsmith's actions
- Build a complete project in seconds
- Iterate and improve on existing work
</Tip>
