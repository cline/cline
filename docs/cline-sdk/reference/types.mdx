All types are exported from their respective packages. This page lists the key types you'll work with.

## Message Types

```typescript
import type { Message, ContentBlock } from "@clinebot/shared"
```

### Message

```typescript
interface Message {
  role: "user" | "assistant" | "system"
  content: ContentBlock[]
}
```

### ContentBlock

Discriminated union of content types:

```typescript
type ContentBlock =
  | TextContent
  | ToolUseContent
  | ToolResultContent
  | ThinkingContent

interface TextContent {
  type: "text"
  text: string
}

interface ToolUseContent {
  type: "tool_use"
  id: string
  name: string
  input: unknown
}

interface ToolResultContent {
  type: "tool_result"
  tool_use_id: string
  content: string | ContentBlock[]
  is_error?: boolean
}

interface ThinkingContent {
  type: "thinking"
  thinking: string
}
```

### AgentResult

```typescript
  text: string
```

### AgentUsage

```typescript

  inputTokens: number

```

### AgentExtension

```typescript
interface AgentExtension {
  name: string
  manifest: {
    capabilities: ("tools" | "hooks" | "commands")[]
  }
  setup?(api: ExtensionAPI): void
  onRunStart?(context: { userMessage: string }): void
  onRunEnd?(context: { result: AgentResult }): void
  onToolCall?(context: { call: ToolCallRecord }): void
  onToolResult?(context: { record: ToolCallRecord }): void
  onIterationStart?(context: { iteration: number }): void
  onIterationEnd?(context: { iteration: number }): void
  onError?(context: { error: Error; recoverable: boolean }): void
}
```

## Hook Types

```typescript
import type { HookEngine } from "@clinebot/shared"
```

### HookPolicy

```typescript
type HookPolicy = "fail_open" | "fail_closed"
```

| Policy | Behavior |
|--------|----------|
| `fail_open` | Hook errors are logged but execution continues |
| `fail_closed` | Hook errors stop the agent |

## Utility Types

### BasicLogger

The logging interface used across the SDK:

```typescript
interface BasicLogger {
  debug(message: string, ...args: unknown[]): void
  info(message: string, ...args: unknown[]): void
  warn(message: string, ...args: unknown[]): void
  error(message: string, ...args: unknown[]): void
}
```

Any logger that implements these four methods works (console, pino, winston, etc.).
