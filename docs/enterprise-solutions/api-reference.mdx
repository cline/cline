This page covers enterprise-focused API endpoints from `api.md`, including user/account management, organizations, member administration, billing, plans, usage, and API key management.

<Note>
  Endpoints below are listed as path references. Use your deployment's API host and standard authorization headers when calling them.
</Note>

## Users

User account management, terms, active account selection, balance, usage, and payments.

| Method | Endpoint |
|---|---|
| `GET` | `/api/v1/users/me` |
| `PATCH` | `/api/v1/users/me` |
| `DELETE` | `/api/v1/users/me` |
| `POST` | `/api/v1/users/me/accept-terms` |
| `GET` | `/api/v1/users/me/remote-config` |
| `PUT` | `/api/v1/users/active-account` |
| `GET` | `/api/v1/users/{id}/balance` |
| `GET` | `/api/v1/users/{id}/usages` |
| `GET` | `/api/v1/users/{id}/payments` |
| `GET` | `/api/v1/users/{id}/payments/{paymentId}` |
| `GET` | `/api/v1/users/{id}/payments/{paymentId}/status` |
| `GET` | `/api/v1/users/{id}/payments/provider/{paymentId}` |
| `POST` | `/api/v1/users/credits/checkout` |
| `POST` | `/api/v1/users/{id}/credits/purchase` |
| `GET` | `/api/v1/users/{id}/auto-top-up` |
| `PUT` | `/api/v1/users/{id}/auto-top-up` |

| `POST` | `/api/v1/users/{id}/payment-method/setup-session` |
| `GET` | `/api/v1/users/{id}/promotions` |

| `DELETE` | `/api/v1/organizations/{id}` |

| `GET` | `/api/v1/organizations/{id}/remote-config` |
