Together AI provides fast inference for popular open-source models with an OpenAI-compatible API.

**Website:** [https://together.ai/](https://together.ai/)

### Getting an API Key

1.  **Sign Up/Sign In:** Go to [Together AI](https://together.ai/). Create an account or sign in.
2.  **Navigate to API Keys:** Access the API key section in your dashboard.
3.  **Create a Key:** Generate a new API key.
4.  **Copy the Key:** Copy the API key immediately and store it securely.

### Configuration in Cline

1.  **Open Cline Settings:** Click the settings icon (⚙️) in the Cline panel.
2.  **Select Provider:** Choose "Together" from the "API Provider" dropdown.
3.  **Enter API Key:** Paste your Together API key.
4.  **Enter Model ID:** Specify the model you want to use.
5.  **Configure Settings:** Optionally set context window size and max output tokens.

### Tips and Notes

-   **OpenAI Compatible:** Together uses an OpenAI-compatible API format.
-   **Wide Model Selection:** Access many open-source models from various providers.
-   **Custom Models:** You can specify any model ID available on Together's platform.
-   **Pricing:** Check the [Together AI pricing page](https://together.ai/pricing) for current rates.
