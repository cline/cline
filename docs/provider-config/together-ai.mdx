Cline supports accessing Deepseek V3 through the Together AI platform.

**Website:** [https://www.together.ai/](https://www.together.ai/)

### Getting an API Key

1.  **Create an Account:** Go to [https://api.together.xyz](https://api.together.xyz) and create an account.
2.  **Get API Key:** Once logged in, navigate to the API key section to get your key.

### Configuration in Cline

1.  **Open Cline Settings:** In the Cline panel, click the settings icon (⚙️).
2.  **Select Provider:** Choose "Together" from the "API Provider" dropdown list.
3.  **Enter API Key:** Paste your Together AI API key into the designated field.
4.  **Enter Model Name:** In the model input field, type `deepseek-ai/DeepSeek-V3`.

That's it! You can now leverage powerful open-source models hosted on Together AI for your development tasks in Cline.
