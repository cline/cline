# XSON VS Code Extension

A VS Code extension created with [create-xson-cline-app](https://github.com/cline/cline/tree/main/packages/create-xson-cline-app).

## Features

- Hello World command

## Development

1. Install dependencies: `npm install`
2. Compile: `npm run compile`
3. Press F5 to run in Extension Development Host

## Publishing

```bash
npm run package
vsce publish
```
