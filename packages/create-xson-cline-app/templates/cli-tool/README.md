# XSON CLI Tool

A CLI tool created with [create-xson-cline-app](https://github.com/cline/cline/tree/main/packages/create-xson-cline-app).

## Installation

```bash
npm install
npm run build
npm link
```

## Usage

```bash
cli-tool hello
cli-tool hello Alice
cli-tool work
```

## Development

```bash
npm run dev
```
